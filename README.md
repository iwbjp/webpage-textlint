# webpage-textlint
Webサイトの特定ページのテキストをtextlintでチェックするサンプル

1. npm installでインストール

```bash
npm install
```

2. ./index.mjs 【URL】で実行

```bash
./index.mjs https://www.dentsu.co.jp/news/release/2019/0206-009744.html
```
